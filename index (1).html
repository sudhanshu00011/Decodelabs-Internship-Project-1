The Toolkit

Your Raw Materials

No frameworks allowed. Master the fundamentals first — HTML5, CSS3, and vanilla JavaScript are your only tools.

🏗️

HTML5 — The Infrastructure

Semantic landmarks and structural meaning. Not just divs — the right element for the right job. Proper use of <header>, <main>, <article>, and <footer>.

🎨

CSS3 — The Skin

Responsive styling using Grid for macro layouts and Flexbox for micro components. Mobile-first with min-width breakpoints at 768px and 1024px.

⚙️

JavaScript — The Logic

Basic state management and interactive functionality. Hamburger menu, scroll events, smooth reveals — no libraries, pure vanilla JS.

📱

Mobile-First Design

Start with a single column, expand outwards. The web is experienced on mobile first — build it that way from line one.

♿

WCAG Accessibility

Strict WCAG adherence. Code for AI and screen readers. Proper ARIA labels, keyboard navigation, and semantic integrity throughout.

✨

2025 UI Aesthetics

Warm, grounded aesthetics using the DecodeLabs 2025 palette — Mocha Mousse, Ethereal Blue, and Moonlit Grey. Max 2 font families, 3 weights.

Execution Plan

The 6-Step Roadmap

Follow this exact sequence to build Project 1 the right way, the professional way.

1

Discovery — Define "How Might We"

Research your user. Create an Empathy Map. We do not guess — we research pain points before writing a single line of code.

2

Wireframe — Grayscale, Mobile-First

Low-fidelity sketches using Visily, Balsamiq, or pen and paper. Golden Rule: if it doesn't work in grayscale, color won't fix it.

3

Semantics — HTML Landmarks

Write the HTML structure first using semantic elements. Proper semantics give screen readers and AI a Table of Contents.

4

Style — CSS Grid / Flex & 2025 Palette

Apply CSS Grid for page-level layout and Flexbox for components. Use the Mocha Mousse, Ethereal Blue, and Moonlit Grey palette.

5

Logic — State & Interactivity

Add vanilla JavaScript for hamburger menus, scroll effects, and any dynamic state management your UI requires.

6

Audit — LCP/CLS & Accessibility

Run Lighthouse. Fix performance metrics (LCP, CLS). Validate WCAG compliance. Only then is your project ready for review.
